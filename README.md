# iitm-netaccess-scripts
Scripts to authenticate through IITM Netaccess

1. Bash:  
    dependencies:  
    - CURL: `sudo apt-get install curl`
    usage:
    - `./netaccess.sh username password`
2. Python:  
    dependencies:
    - requests: `pip install requests`  
    usage:
    - `./netaccess_commandline.py`
